반품신청

1단계 : 반품신청

  • 반품은 [마이페이지 > 나의쇼핑내역 > 주문/배송조회 또는 취소/반품조회] 에서 신청하실 수 있습니다.
  • 반품은 상품 수령하고 주문상태가 배송완료가 된 후, 반품신청을 통해 가능 합니다.
  • 상품의 택을 제거하였거나 신발 박스가 훼손 되었을 경우 반품이 불가능합니다.

2단계 : 반품배송

  • 반품신청 후 3~5일 내에 입력한 주소로 택배 기사가 방문하여 상품을 회수 합니다.
  • 배송 위탁 계약사인 한진택배 외 다른 택배회사를 이용하여 반품하는 경우 반품이 지연될 수 있으며, 분실 시 보상이 불가능합니다.
  • 부득이 하게 다른 택배 회사를 이용하였을 경우 반드시 고객센터로 연락을 주셔야 원활한 반품이 진행됩니다.

3단계 : 상품 상태 확인

  • 회수되어 매장에 도착한 상품은 상태 확인 후 이상이 없을 시 반품완료 처리 됩니다.
  • 상품 훼손이나 반품 시 선택한 사유와 상이하여 이상이 있을 경우 콜센터에서 연락을 드리며, 환불이 지연되거나 불가능할 수 있습니다.

4단계 : 반품 환불

  • 반품 금액은 사유에 따라 배송비를 차감한 금액만큼 환불 됩니다.
  • 신용카드 및 체크카드 : 영업일 기준 약 2~3일 후 카드승인 취소
  • 실시간 계좌 이체 : 영업일 기준 약2~3일 후 이체 진행 계좌로 환불
  • 무통장입금 : 영업일 기준 약1~2일 후 반품 시 입력한 계좌로 환불
교환신청

1단계 : 교환신청

  • 교환은 [마이페이지 > 나의쇼핑내역 > 주문/배송조회] 에서 주문상태가 배송완료가 된 후, 교환신청을 통해 가능 합니다.
  • 교환은 동일한 제품의 사이즈 변경만 가능하며, 다른 제품으로 교환하실 경우 반품 후 재주문 해주셔야 합니다.
  • 교환은 상품의 택을 제거하였거나 신발 박스가 훼손 되었을 경우 교환이 불가능합니다.

2단계 : 교환배송

  • 교환신청 후 3~5일 내에 입력한 주소로 택배 기사가 방문하여 상품을 회수 합니다.
  • 배송 위탁 계약사인 한진택배 외 다른 택배회사를 이용하여 교환하는 경우 교환이 지연될 수 있으며, 분실 시 보상이 불가능합니다.
  • 부득이 하게 다른 택배 회사를 이용하였을 경우 반드시 고객센터로 연락을 주셔야 원활한 교환이 진행됩니다.

3단계 : 상품 상태 확인

  • 회수되어 매장에 도착한 상품은 상태 확인 후 이상이 없을 시 상품상태 확인이 완료되며, 교환 배송비 결제 안내 SMS가 발송 됩니다.
  • 상품 훼손이나 교환 시 선택한 사유와 상이하여 이상이 있을 경우 콜센터에서 연락을 드리며, 교환이 지연되거나 불가능할 수 있습니다.

4단계 : 교환 배송비 결제

  • 교환 배송비는 로그인 시에 생성되는 페이지 혹은 [마이페이지 > 나의쇼핑내역 > 교환결제대기] 에서 결제가 가능합니다.
  • 배송비는 교환 사유에 따라 책정되며, 제주도 및 도서산간 지역은 배송비가 추가될 수 있습니다.

5단계 : 교환 상품 발송

  • 교환 배송비 결제가 완료되면 교환 요청한 상품이 발송 되며, 발송 전 1~2일 정도의 기간이 소요될 수 있습니다.
  • 매장 재고 상황에 따라 최초 배송 매장이 아닌 타매장에서 발송 될 수도 있습니다.
